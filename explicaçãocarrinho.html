Explicação do Projeto: Carrinho Seguidor de Linha

O objetivo deste projeto é desenvolver um carrinho seguidor de linha utilizando a plataforma Arduino. O processo envolve a construção física do carrinho, a programação do Arduino para controlar os motores e sensores, e a documentação do processo em um relatório técnico.

Relatório:
O projeto foi iniciado com a montagem do motor, seguido pela construção da roda traseira. A fiação foi realizada conectando a bateria, os sensores e os motores, e o código foi programado para controlar o funcionamento do carrinho.

Materiais utilizados:
Os componentes principais do projeto são:

  • Kit de robótica Arduino (ou componentes individuais: placa Arduino, motores, rodas, chassi, sensores de linha, etc.)
  • Software Arduino IDE
  • Ferramentas de solda (opcional)
  • Tinkercad
  • Câmera ou celular para registro de fotos e vídeos

Desenvolvimento do Projeto:
A construção do carrinho seguidor de linha envolveu várias etapas, desde a montagem do motor até a instalação dos sensores de linha e a programação do Arduino para controlar os movimentos.

Conclusão:
Este projeto demonstrou a importância do trabalho em equipe e da aplicação dos conhecimentos adquiridos ao longo do curso. A prática com componentes eletrônicos e a programação de um sistema autônomo proporcionaram uma valiosa experiência.
