Explicação do Projeto

O objetivo do projeto é criar um sistema de acessibilidade para pessoas com deficiência visual, através de um boné com acessórios complementares para aumentar a interatividade e o auxílio ao usuário.

Relatório:
Iniciamos o desenvolvimento do projeto projetando a estrutura e o sistema no Tinkercad, utilizando uma protoboard para montar os componentes necessários. O Arduino Uno foi escolhido para controlar os dispositivos eletrônicos e programar o funcionamento do sistema.

Componentes utilizados:
Os principais componentes conectados à protoboard são:

  • 1 Buzzer
  • 1 Sensor de Distância Ultrassônico (HC-SR04)
  • 3 LEDs
  • 5 Capacitores de 0,3k
  • 1 Sensor LDR (Fotovoltaico)
  • 9 Cabos Jumper

Origem do nome Arduino:
O nome "Arduino" foi inspirado pelo nome de um bar na cidade de Ivrea, na Itália, onde os criadores do projeto se reuniam para desenvolver a ideia que se tornou o Arduino.

Explicação do Código:
O código foi desenvolvido durante o bimestre. Inicialmente, declaramos as variáveis correspondentes aos pinos do Arduino e configuramos os pinos como OUTPUT (para os dispositivos que recebem comandos) e INPUT (para os sensores que armazenam dados).

A programação do sensor ultrassônico utiliza a fórmula duração * 0.034 / 2 para calcular a distância. A condição if (distance > 0 && distance <= 40) ativa o LED vermelho e o buzzer se a distância for menor que 40 cm, enquanto a condição else (distance > 40 && distance <= 100) acende o LED verde e desliga o buzzer.

O código do sensor LDR foi implementado usando um valor booleano para determinar a intensidade da luz. Utilizamos a lógica de if e else para controlar o LED amarelo, ligando-o ou desligando-o conforme o valor lido.

Relacionamento com Outras Áreas:
Matemática: A matemática é essencial para o cálculo da distância com o sensor ultrassônico. A fórmula utilizada envolve multiplicação e divisão.
Linguagens: Os LEDs são usados como uma forma de comunicação visual, com o LED vermelho indicando "PARA!" e o verde "OK!".
Ciências Humanas: Este projeto pode beneficiar pessoas com deficiência visual, ajudando a melhorar sua mobilidade e autonomia.
Ciência da Natureza: O sistema pode ser aplicado para monitoramento de ambientes e em projetos de automação, como no controle de irrigação na agricultura.
